========================================
README - Website Sejarah dan Fun Facts Starbucks
========================================

Deskripsi:
-----------
Website ini dibuat untuk memberikan informasi mengenai sejarah dan fakta menarik tentang Starbucks. Menggunakan desain bergaya Starbucks, website ini menawarkan pengalaman yang nyaman dan menarik bagi pengunjung.

Struktur Halaman:
-----------------
1. **Landing Page**:
   - Berisi nav bar untuk  pergi ke feedback atau content utama

2. **Bagian Utama (Main Content)**:
   - **Sejarah Starbucks**:
     Menjelaskan perjalanan Starbucks dari awal berdiri pada tahun 1971 hingga menjadi brand kopi global dengan lebih dari 35.000 gerai di seluruh dunia.
   - **Fun Facts tentang Starbucks**:
     Menampilkan fakta-fakta unik tentang Starbucks dalam format daftar (list).

3. **Feedback**:
   - Pengunjung dapat mengisi feedback dengan mengisi 2 format yaitu nama dan isi feedback.

Fitur :
-------------
- **Desain Responsif**:
  Halaman akan menyesuaikan tampilannya pada berbagai ukuran layar, baik di desktop maupun perangkat mobile.
  
- **Gaya Starbucks**:
  Menggunakan warna-warna khas Starbucks seperti hijau (#00704A) dan krem (#f3f1e7) untuk menciptakan nuansa khas kedai kopi ini.

- **Ubah Tema**:
  Di halaman landing page kita dapat megubah tema bernuansa gelap atau starbucks.
